# Button 按钮

按钮用于开始一个即时操作。

## 演示例子

<demo src="../../demos/button/basic.vue"></demo>


## Api
| 属性 | 说明 | 类型 | 默认值 |
| - |:--:| :-:| -:|
| type | 类型 | `default, primary, success, warning, danger, link` | default |
| size | 尺寸 | `normal, small, large` | normal |

## Slot
| 插槽名 | 说明 |
| - |:--:|
| default | 默认加载内容 |
